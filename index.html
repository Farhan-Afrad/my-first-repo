For 1st commit of a file 6 things should be done

  1. git init
  2. git add .
  3. git commit -m "message"
  4. git branch -M main
  5. git remote add origin https://.......
  6. git push -u origin main

After 1st commit and after chaning somethings 3 things should be done

  1. git add .
  2. git commit -m "message"
  3. git push
